## モバイルカレンダー
### 達成したこと
カレンダーの予定から往訪先を推定するモデルを作るために、教師データを登録するWebアプリケーションを実装した。
ユーザが投稿した教師データを登録するためのWebAPIをElixirで実装した。

### 達成できなかったこと
SPEEDAモバイルの開発が中止になったので、集めた教師データを使い、モデルを実装することはなかった。

## トレンドとニュースの紐づけ
### 達成したこと
モデルを訓練するプログラムをK8S環境で動かすパイプラインとトレンドとニュースを紐づけるWeb APIを実装した。

### 達成できなかったこと
予測性能を現行のクエリによる紐付け以上にあげられず、モデルをリリースできなかった。
Weak supervisionをはじめとする技術調査とその試行で作業を中断している。
